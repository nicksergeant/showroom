Showroom UI
===========

The UI for [Showroom](http://showroom.is).

### Install dependencies:

1. `npm install -g browserify jsmin react-tools`
2. `brew install fswatch`

### Run and develop locally:

1. `make run`

### Compile:

2. `make compile`

### Watch:

1. `brew install fswatch`
2. `make watch`
